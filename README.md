### Описание
Проект состоит из реализации консольной программы для генерации лабиринтов и поиска пути в них.
Алгоритмы генерации лабиринтов:
1. Алгоритм DFS
2. Алгоритм Прима
3. Алгоритм бинарного дерева
4. Алгоритм Олдоса-Бродера

Алгоритмы поиска решения (при отсутствии решения выводится лабиринт без решения):
1. A-Star
2. Алгоритм Дейкстры
3. Волновой алгоритм

**Генерация лабиринтов.**
```
-a / --algorithm - алгоритм генерации (dfs, prim, binarytree, oldosbroder)
-w / --width - ширина лабиринта (итоговые размеры лабиринта - (2height +1) на (2width+1))
-h / --height - высота лабиринта (итоговые размеры лабиринта - (2height +1) на (2width+1))
 --output - опционально можно выбрать путь для вывода
-u / --unicode - опционально можно выводить unicode символы
```
Пример:
```bash
java -jar target/project-1.0.jar generate -a=dfs -w=15 -h=30  -u=true
```

**Решение лабиринтов.**

```
-a / --algorithm - алгоритм для решения(astar, dijkstra, wave)
--file - название файла с лабиринтом
-s / --start - координаты начальной точки в формате x,y ( 0<x<2height +1, 0<y<2width+1)
-e / --end - координаты конечной точки в формате x,y ( 0<x<2height +1, 0<y<2width+1)
--output - опционально можно выбрать путь для выводв
-u / --unicode -  опционально можно выводить unicode символы
-w / --weight - опционально можно решать лабиринты со случайными весами полей. 
В таком случае случайно удалится часть стен, чтобы поиск оптимального пути имел смысл.
```
Пример:
```bash
java -jar target/project-1.0.jar solve --algorithm=astar --file=maze.txt --start=1,1 --end=9,9  -u=true
```
